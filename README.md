Vote Checker App (Java GUI)

A simple Java desktop application that checks if a user is eligible to vote based on their age.

🚀 Features
Enter user name
Enter age
Select gender (Male/Female)
Check voting eligibility
Displays result instantly
Input validation (no empty fields, valid age required)
🖥️ Built With
Java (JDK 17+)
Java Swing (GUI)
NetBeans IDE
📸 How It Works
Enter your name
Enter your age
Select your gender
Click Check Vote
View result:
✅ Eligible to vote (18 and above)
❌ Not eligible (below 18)
📂 Project Structure
voteChecker/
 ├── src/
 │    └── VoteCheckerGUI.java
 ├── nbproject/
 ├── build.xml
▶️ How to Run
Open project in NetBeans
Run the project
The application window will appear

OR using terminal:

javac VoteCheckerGUI.java
java VoteCheckerGUI
